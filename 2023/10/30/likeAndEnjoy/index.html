Like 和 Enjoy

2023-10-30- 2023-11-01
英语-单词记忆固定搭配

Like

Like v. 喜欢

like 可以加不定式，表示喜欢去做某事，to 在这里代表“去做”。

like to do sth

从不定式的名字上就能看出，这件事还没有定，比如

I like to play basketball

我喜欢玩篮球这件事还没有定，也就是我不总是喜欢打篮球，说不定我现在想打，明天又不想打了，是不确定的，所以不定式是表示短期的，或者没有确定的事

其他带有不定式的搭配，比如

want to do sth

想要去做某事，但这件事是想要去做，但还没有做，说不定你现在想要去做某事，一会儿就不想做了。所以是不确定的。to 在这里还是表示“去做”

如果没有体现不确定性，说明是长期的喜欢。like 后直接加动词即可（like do），但直接加动词一个句子里就有两个谓语了，这在英语语法上是不允许的（like do），所以要将 like 后面的动词加上 ing 形式。变为 like doing。

总结：

持续喜欢做某事 喜欢去做某事
like doing like to do
我之前喜欢，现在喜欢，以后还喜欢 我就现在喜欢，说不定明天就不喜欢了
书上的解释： （仅供了解，不用背）
对特定动作的享受或乐趣 对特定动作的偏好或倾向
✅ Do this ❌ Don’t do this
like to do like to doing（不定式用法错误）
like doing like do（语法错误）

Enjoy doing

enjoy v.享受

你想，你都享受做某事了，肯定已经定了吧，所以没有不定式的用法，记住 enjoy doing 就好

✅ Do this ❌ Don’t do this
enjoy doing enjoy to do

Practice makes perfect

判断：

  1. Xiao Ming likes playing sports every day (T/F)
  2. Ma Yongxin likes to play soccer every day (T/F)
  3. I like learn English (T/F)

填空：
4. He enjoys ___ (play) basketball very much

  1. You like ___ (do) exercises since you was a little boy

    ( 从你小时起，你就喜欢锻炼了 )

答案：

点击查看 判断1-3：TFF、填空4-5：playing；doing

解析：
1.小明每天都喜欢做运动，是长期性的，所以 doing 是对的
2.马永鑫每天都喜欢踢足球，同上，是长期性，to do 是短期的，所以错了
3.这句话没有明确是长期还是短期，所以 to do和doing 都对，但他写错了，所以错了
4.enjoy只能接 doing
5.从你小时起表示了长期，所以填 doing
